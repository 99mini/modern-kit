# When

condition prop이 `true`면 자식 컴포넌트를 렌더링하고, `false`면 렌더링하지 않는 컴포넌트입니다.

condition prop으로 `boolean을 반환하는 함수`도 허용됩니다.

<br />

## Interface
```tsx
type Condition = boolean | ((...args: any[]) => boolean);

interface WhenProps {
  condition: Condition;
}

const When: ({ children, condition }: PropsWithChildren<WhenProps>) => JSX.Element | null
```

## Usage

```tsx
import { When } from '@modern-kit/react';
import { Button } from '@modern-kit/ui';

const Example = () => {
  const [state, setState] = useState(false);

  return (
    <>
      <Button onClick={() => setState(!state)}>Toggle Button</Button>

      <When condition={state}>
        <h1>Render</h1>
      </When>
    </>
  );
};
```
